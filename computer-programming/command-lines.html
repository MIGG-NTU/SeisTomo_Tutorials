Command Lines¶

Seismologists usually use command lines (CLI) rather than Graphic User Interface (GUI) to navigate around different folders/directories, run programs, process data and almost everything you can imagine, because using CLI is usually much efficient than using GUI.

Data Processing¶

awk¶

awk is a domain-specific language designed for text processing and typically used as a data extraction and reporting tool. It is very convenient to extract certain columns in an ASCII-format file using awk given a specified division character, e.g., empty space ” ” as default, comma “,”, semicolon “;”, dash “-“, or other describable characters. Here are two expamles showing the way that we usually use to extract location, magnitude, and origin time from a coomon catalogue file.

# check a catalog file, e.g., events.csv
$ cat events.csv
2019-09-01T10:30:32.320Z  32.398   90.841  39.32  6.5  MW
2013-03-11T12:21:01.149Z  19.691  120.933  23.83  4.5  MW
2010-01-09T01:01:10.921Z  51.582  110.850   9.33  3.3  MB

# extract the depth (fourth) and magnitude (fifth) columns
$ awk '{print $4,$5}' events.csv
39.32 6.5
23.83 4.5
9.33 3.3

# extract the origin time (first) column and reformat into yyyymmdd hhmmsss
$ awk '{print $1,$5}' events.csv
2019-09-01T10:30:32.320Z 6.5
2013-03-11T12:21:01.149Z 4.5
2010-01-09T01:01:10.921Z 3.3
